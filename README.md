# Al-Kawthar Translation Dataset

## Overview

This dataset documents comparative English translations of selected aḥādīth from Ṣaḥīḥ al-Bukhārī concerning **al-Kawthar** (Q 108:1) and **al-ḥawḍ** (the Basin), generated as part of a doctoral research project on the application of large language models (LLMs) to Islamic textual corpora.

The dataset compares three translation approaches:
- **Muḥsin Khān** (human translator): the widely used English translation of Ṣaḥīḥ al-Bukhārī, characterised by an interpretive-exegetical approach
- **ChatGPT-5 (Thinking mode)**: OpenAI's reasoning-enabled model
- **DeepSeek (DeepThink mode)**: DeepSeek's reasoning-enabled model

## Dataset Contents

The file `al-kawthar-translations.csv` contains the following columns:

| Column | Description |
|--------|-------------|
| `Arabic Text` | Original Arabic text of the ḥadīth |
| `Hadith Reference` | Book and hadith number in Ṣaḥīḥ al-Bukhārī |
| `Muhsin Khan Translation` | Human translation (interpretive-exegetical approach) |
| `ChatGPT-5 Thinking Translation` | LLM output under reasoning-enabled configuration |
| `DeepSeek DeepThink Translation` | LLM output under reasoning-enabled configuration |

## Methodology

All LLM outputs were generated during testing conducted in **October 2025**. Models were prompted with a constrained instruction set explicitly requesting direct, faithful translation devoid of added interpretation, commentary, or doctrinal notes.

The full prompt specifications used in the experiments are documented in the associated doctoral dissertation (University of Palermo / University of Modena and Reggio Emilia).

## Research Context

This dataset was produced as part of doctoral research on generative AI and large language models applied to Islamic textual corpora, with particular focus on AI-induced errors, bias, and grounding/verification methods for sensitive religious content. The research is conducted within the framework of the **Digital Maktaba Project**.

Key analytical findings documented using this dataset include:
- Systematic differences in terminological choices between human and AI translators (e.g. al-ḥawḍ rendered as "Basin" by AI models vs "Fount," "Tank," or "Lake-Tank" by Muḥsin Khān)
- Divergent handling of isnād chains
- Differences in the treatment of implicit references and contextual insertions
- Stylistic divergence between ChatGPT-5 and DeepSeek in rendering key terms such as faraṭ ("forerunner" vs "predecessor")
